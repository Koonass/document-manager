================================================================================
     DOCUMENT MANAGER V2.4 - PORTABLE EDITION
     Quick Start Guide
================================================================================

CONGRATULATIONS! You have the portable version - no installation needed!

================================================================================
  STEP 1: COPY THE FOLDER
================================================================================

Copy this entire folder to one of these locations:

  [RECOMMENDED] USB Drive:
  E:\DocumentManager\  (or F:\, G:\, etc. - your USB drive letter)

  OneDrive:
  C:\Users\YourName\OneDrive\Apps\DocumentManager\

  Local Drive:
  C:\Apps\DocumentManager\

  Network Share:
  \\server\share\DocumentManager\

IMPORTANT: Copy the ENTIRE folder, not just some files!

================================================================================
  STEP 2: RUN THE APPLICATION
================================================================================

Double-click:  START_PORTABLE.bat

That's it! The application will start immediately.

(First time only: It will ask to create settings and folders - say Yes)

================================================================================
  DAILY USE
================================================================================

Just double-click:  START_PORTABLE.bat

The application remembers everything from last time.

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Portable Python not found!"
Solution:
  - Make sure you copied the ENTIRE folder (including python\ subfolder)
  - If on OneDrive, wait for sync to complete (green checkmark)
  - Check folder size: should be ~350-400 MB

Problem: Application won't start
Solution:
  - Check antivirus didn't block it
  - Try copying to C:\Apps\ instead
  - See PORTABLE_DEPLOYMENT_GUIDE.md for detailed help

Problem: Database locked
Solution:
  - Close other instances of the app
  - If on network/OneDrive, only 2-3 users can run simultaneously
  - Close the app when not using it
