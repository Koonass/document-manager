WORKFLOW
================================================================================

1. User imports Bistrack CSV file
2. PDF Processor scans PDF folder
   ├─ Tries to extract sales order from filename
   └─ Falls back to PDF content extraction
3. Sales orders matched against CSV records
4. Relationship Manager creates order-PDF links
5. Enhanced Database stores relationships + history
6. Calendar displays unprocessed items (Mon-Fri only)
7. User clicks date → Expanded view shows details
8. Items categorized:
   ├─ Green: Ready to print (has PDF)
   ├─ Red: Missing PDF
   └─ Gray: Already processed
9. User can:
   ├─ View/download PDF links
   ├─ Attach missing PDFs
   ├─ Batch print all
   └─ Auto-generate labels
10. Archive Manager moves processed items to archive
11. Database marks as processed
12. Items disappear from calendar
